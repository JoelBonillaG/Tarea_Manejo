Página Elecciones UTA

  • Tecnologías Empleadas: Laravel, PHP, CSS, HTML, JavaScript, Bootstrap, MySQL
  • Repositorio: Enlace al Proyecto

La Página Elecciones UTA fue desarrollada como una plataforma digital para apoyar la campaña de la Dra. Mary Cruz Lascano durante las elecciones rectorales de la Universidad Técnica de Ambato, periodo 2024-2028. Esta página tuvo como objetivo central informar y conectar a los estudiantes, docentes y personal administrativo con las propuestas del partido.

La página incluyó una sección dedicada a presentar información detallada de cada miembro del equipo de campaña, junto con sus propuestas, eventos programados y noticias relevantes. Además, ofreció un apartado de estadísticas en tiempo real para mostrar el apoyo que recibió el partido durante la campaña, brindando una visión transparente y actualizada del impacto del proyecto.

Este proyecto fue desarrollado utilizando el framework **Laravel** para una gestión eficiente de datos, complementado con **Bootstrap** para un diseño responsivo y moderno, y **MySQL** como base de datos para un manejo estructurado de la información. La combinación de estas tecnologías aseguró que la plataforma fuera funcional, atractiva y fácil de usar.
