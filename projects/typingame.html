Typing Game

  • Tecnologías Empleadas: Java
  • Repositorio: Enlace al Proyecto

Typing Game es un juego local diseñado para poner a prueba tu velocidad y precisión al escribir en el teclado. Ideal para sesiones de juego entre dos o más jugadores, el teclado se divide en dos mitades, asignando un lado a cada jugador para competir simultáneamente.

En cada ronda, se muestran palabras aleatorias en la pantalla que deben ser tipeadas rápidamente por los jugadores. Los errores cometidos durante la escritura se registran y afectan el puntaje final. Tras un número definido de rondas, el jugador con menor desempeño queda eliminado, avanzando los demás a una siguiente fase de clasificación.

Al finalizar el torneo, se presentan los tres primeros lugares junto con sus puntajes, destacando a los mejores jugadores. Este proyecto combina competencia y diversión, ofreciendo una experiencia desafiante para mejorar las habilidades de escritura.
